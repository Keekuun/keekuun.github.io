## Context

```js
const gl = canvas.getContext('webgl');       // WebGL 1
const gl2 = canvas.getContext('webgl2');     // WebGL 2（向下兼容概念，API 有扩展）

gl.viewport(x, y, width, height);
gl.clearColor(r, g, b, a);
gl.clear(mask);  // COLOR_BUFFER_BIT | DEPTH_BUFFER_BIT | STENCIL_BUFFER_BIT
```

| 常量 | 值含义 |
|------|--------|
| `COLOR_BUFFER_BIT` | 清空颜色 |
| `DEPTH_BUFFER_BIT` | 清空深度 |
| `STENCIL_BUFFER_BIT` | 清空模板 |

## 着色器与 Program

```js
const vs = gl.createShader(gl.VERTEX_SHADER);
const fs = gl.createShader(gl.FRAGMENT_SHADER);
gl.shaderSource(vs, source);
gl.compileShader(vs);
gl.getShaderParameter(vs, gl.COMPILE_STATUS);
gl.getShaderInfoLog(vs);

const program = gl.createProgram();
gl.attachShader(program, vs);
gl.attachShader(program, fs);
gl.linkProgram(program);
gl.getProgramParameter(program, gl.LINK_STATUS);
gl.getProgramInfoLog(program);

gl.useProgram(program);
gl.deleteShader(vs); // link 后可删
```

| 类型 | 常量 |
|------|------|
| 顶点着色器 | `VERTEX_SHADER` |
| 片段着色器 | `FRAGMENT_SHADER` |

---

## Buffer 与 Attribute

```js
const buf = gl.createBuffer();
gl.bindBuffer(gl.ARRAY_BUFFER, buf);
gl.bufferData(gl.ARRAY_BUFFER, data, gl.STATIC_DRAW);

const loc = gl.getAttribLocation(program, 'a_position');
gl.enableVertexAttribArray(loc);
gl.vertexAttribPointer(loc, size, type, normalize, stride, offset);

// 索引
gl.bindBuffer(gl.ELEMENT_ARRAY_BUFFER, indexBuf);
gl.drawElements(gl.TRIANGLES, count, gl.UNSIGNED_SHORT, offset);
```

| `bufferData` usage | 含义 |
|--------------------|------|
| `STATIC_DRAW` | 设一次，画很多次 |
| `DYNAMIC_DRAW` | 偶尔改 |
| `STREAM_DRAW` | 每帧改 |

| `type` | 对应 JS 类型 |
|--------|-------------|
| `FLOAT` | Float32Array |
| `UNSIGNED_BYTE` | Uint8Array |
| `UNSIGNED_SHORT` | Uint16Array |

---

## Uniform

```js
const loc = gl.getUniformLocation(program, 'u_matrix');
gl.uniform1f(loc, v);
gl.uniform2fv(loc, [x, y]);
gl.uniform3fv(loc, [x, y, z]);
gl.uniform4fv(loc, [x, y, z, w]);
gl.uniformMatrix4fv(loc, false, array16);
gl.uniform1i(samplerLoc, textureUnitIndex);
```

矩阵 `uniformMatrix4fv` 第二参数 `transpose` 在 WebGL 里 **必须为 false**。

## 纹理

```js
const tex = gl.createTexture();
gl.activeTexture(gl.TEXTURE0 + unit);
gl.bindTexture(gl.TEXTURE_2D, tex);
gl.texImage2D(gl.TEXTURE_2D, 0, gl.RGBA, width, height, 0, gl.RGBA, gl.UNSIGNED_BYTE, data);
gl.texParameteri(gl.TEXTURE_2D, gl.TEXTURE_MIN_FILTER, gl.LINEAR);
gl.texParameteri(gl.TEXTURE_2D, gl.TEXTURE_WRAP_S, gl.CLAMP_TO_EDGE);
gl.generateMipmap(gl.TEXTURE_2D);

gl.pixelStorei(gl.UNPACK_FLIP_Y_WEBGL, true);
```

| 参数 | 常用 |
|------|------|
| `TEXTURE_MIN_FILTER` | `LINEAR`, `LINEAR_MIPMAP_LINEAR`, `NEAREST` |
| `TEXTURE_MAG_FILTER` | `LINEAR`, `NEAREST` |
| `TEXTURE_WRAP_S/T` | `CLAMP_TO_EDGE`, `REPEAT` |

## 绘制

```js
gl.drawArrays(mode, first, count);
gl.drawElements(mode, count, type, offset);
```

| `mode` | 图元 |
|--------|------|
| `TRIANGLES` | 三角形列表 |
| `LINES` | 线段 |
| `POINTS` | 点 |
| `TRIANGLE_STRIP` | 三角带 |

## 帧缓冲

```js
const fb = gl.createFramebuffer();
gl.bindFramebuffer(gl.FRAMEBUFFER, fb);
gl.framebufferTexture2D(gl.FRAMEBUFFER, gl.COLOR_ATTACHMENT0, gl.TEXTURE_2D, tex, 0);
gl.framebufferRenderbuffer(gl.FRAMEBUFFER, gl.DEPTH_ATTACHMENT, gl.RENDERBUFFER, rb);
const ok = gl.checkFramebufferStatus(gl.FRAMEBUFFER) === gl.FRAMEBUFFER_COMPLETE;
gl.bindFramebuffer(gl.FRAMEBUFFER, null); // 画回屏幕
```

| `checkFramebufferStatus` | 含义 |
|--------------------------|------|
| `FRAMEBUFFER_COMPLETE` | 可用 |
| `FRAMEBUFFER_INCOMPLETE_ATTACHMENT` | 附件无效 |
| `FRAMEBUFFER_INCOMPLETE_DIMENSIONS` | 尺寸不匹配 |
| `FRAMEBUFFER_UNSUPPORTED` | 格式组合不支持 |

详见 [15 · FBO](./15-framebuffer.md)。

## 固定管线状态

```js
// 深度
gl.enable(gl.DEPTH_TEST);
gl.depthFunc(gl.LESS);
gl.depthMask(true);

// 面剔除
gl.enable(gl.CULL_FACE);
gl.cullFace(gl.BACK);
gl.frontFace(gl.CCW);

// 混合（透明）
gl.enable(gl.BLEND);
gl.blendFunc(gl.SRC_ALPHA, gl.ONE_MINUS_SRC_ALPHA);

// 多边形偏移（减少 Z-fighting）
gl.enable(gl.POLYGON_OFFSET_FILL);
gl.polygonOffset(factor, units);
```

## 错误与调试

```js
const err = gl.getError();
// gl.NO_ERROR, gl.INVALID_ENUM, gl.INVALID_VALUE, gl.INVALID_OPERATION, gl.OUT_OF_MEMORY
```

| 扩展 | 用途 |
|------|------|
| `WEBGL_lose_context` | 测试 context lost |
| `OES_element_index_uint` | WebGL1 用 32 位索引 |
| `ANGLE_instanced_arrays` | WebGL1 实例化（WebGL2 内置） |

常用查询：

```js
gl.getParameter(gl.MAX_TEXTURE_SIZE);
gl.getParameter(gl.MAX_VERTEX_ATTRIBS);
gl.getSupportedExtensions();
```

调试流程见 [16 · 调试与工具](./16-debugging.md)。
